HAPI 1.1 server verifier. See server-verifier-nodejs for source code for tests.

Usage: ?url=URL&id=ID&parameter=PARAMETER&time.min=TIME&time.max=TIME&id=ID.

Only a URL that end in /hapi is required. Only one PARAMETER may be specified (comma-separated list not supported).

Default is to use time.min=startDate and time.max=startDate+P1D and check all dataset IDs from /catalog response.

Examples

  • ?url=http://mag.gmu.edu/TestData/hapi&time.min=1970-01-01&time.max=1970-01-01T00:00:10&id=dataset1&parameter=scalar
  • ?url=http://mag.gmu.edu/TestData/hapi&time.min=1970-01-01&time.max=1970-01-01T00:00:10&id=dataset1
  • ?url=http://mag.gmu.edu/TestData/hapi&time.min=1970-01-01&time.max=1970-01-01T00:00:10
  • ?url=http://mag.gmu.edu/TestData/hapi&time.min=1970-01-01
  • ?url=http://mag.gmu.edu/TestData/hapi
